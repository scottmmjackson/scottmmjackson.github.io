The following demo parses a first-order differential equation and approximates it with Euler's Method

If you're wondering why this app takes so long to load, it's because equation parsing is now supported using the mathjs library. Please see the documentation page for details on how equations are interpreted.

The table was created with d3.js. While it's possible to do it without d3, I've been trying to push myself to use d3 more often.
